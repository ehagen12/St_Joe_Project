AfghanistanData <- subset(gap.data.in, country== "Afghanistan")
BurundiData <- subset(gap.data.in, country== "Burundi")

THIS IS THE SECTION OF SCATTER PLOTS: In Afghanistan you see a trend downward as you go into the future.

ggplot(data=AfghanistanData, aes(x=year, y=gdpPercap)) + geom_point(aes(color=country, shape=country), size =6) + geom_smooth(method=lm)

However, in Burundi you see a trend upward in GDP per Cap as the years go forward.

ggplot(data=BurundiData, aes(x=year, y=gdpPercap)) + geom_point(aes(color=country, shape=country), size =6) + geom_smooth(method=lm)

Finally, In Canada you see an even greater increase in GDP per Cap.

ggplot(data=CanadaData, aes(x=year, y=gdpPercap)) + geom_point(aes(color=country, shape=country), size =6) + geom_smooth(method=lm)

Note that the echo = FALSE parameter was added to the code chunk to prevent printing of the R code that generated the plot.
